Getting Started ​

  1. Install the latest release
  2. Open the QMK MSYS shortcut
  3. Run qmk setup
  4. Run qmk compile
  5. Profit!

Next Steps ​

The QMK Tutorial will walk you through creating a keymap for your keyboard.
