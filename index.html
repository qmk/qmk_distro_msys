QMK MSYS

A Windows one-click installer for the QMK CLI

Get Started → Latest Version ↓

Want to get started with QMK Firmware (opens new window) quickly? Running Windows 8/10 and want to avoid installation headaches? Look no further!

Bundling MSYS2 + QMK CLI together, this installer package aims to give you a QMK build environment with as little effort as possible.

Not using Windows? Go directly to the Tutorial (opens new window) to get started.


This project is maintained by QMK collaborators and contributors like you!
